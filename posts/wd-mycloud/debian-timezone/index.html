Change Timezone in Debian

2020-05-17 ::
#WD MyCloud  #Debian  #Timezone  #timedatectl 

Using timedatectl⌗

Current Timezone⌗

$ timedatectl

Set Timezone⌗

$ timedatectl set-timezone Asia/Kolkata

List Timezones⌗

$ timedatectl list-timezones
$ timedatectl list-timezones | grep -i kolkata
