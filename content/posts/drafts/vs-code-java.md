Installing OpenJDK 8 on j
```
sudo apt-get install openjdk-8-jdk
ls /usr/lib/jvm/
```

Set JAVA_HOME
```
sudo nano /etc/environment
JAVA_HOME="/usr/lib/jvm/java-8-openjdk-amd64"
```
