Developing an Angular project on Cloud9 IDE

2018-07-10
#Angular  #Node.js  #NPM  #Tutorial  #Cloud9 IDE 
Update NodeJS version All of the blank VMs on cloud9 have nvm pre installed. List all the available versions from the official site. nvm ls-remote Choose to install the latest version nvm install v10.6.0 Choose to use the latest version nvm use v10.6.0 Check the version in use node –version Angular Install latest Angular CLI package npm install -g @angular/cli Create a new project named my-project (you can use any name you need) ng new my-project cd my-project Change start script in package.
